# portifolio
Repositório destinado para portifolio pessoal


Crie um site de protfolio pessoal moderno e responsivo usando, HTML, CSS e Bootstrap 5.
Tema: visual: "Desenvolvedora de Sistemas..."
Paleta de cores: "branco preto"
Estilo do desing: "profissional" 

O site deve ter:
- Navar fixa e responsiva
- Seção hero com foto, nome e frase de destaque
- Seção "Sobre Mim"
- Seção de proejtos com imagens e descrição (2 projetos)
- Seção de experiencia ou estudos
- Seção de contatos com formulário
- Footer moderno com redes sociais

Adicione:
- Animações suaves ao rola a página 
- Efeitos hover nos botões e cards
- Ícones usando Bootstrap Icons ou Font Awesome
- Layout totalmente responsivo para celular e desktop
- Scroll suave entre seções
- Design moderno e profissional

Tecnologias obrigatórias:
- Html5
- CSS3
- Bootstrap 5
- JavaScript opcional para interações

Organize os arquivos em:
- index.html
- style.css
- script.js

Use código limpo, organizado e comentado.
Crie um visual impressionante e moderno, parecido com sites profissionais de desenvolvedores.
